# Required software for Hands-on session

## For Python

For the hands-on part of this week, I have one brief example using data from the Scholarly Migration Database in Python.

Please use the provided `requirements.txt` file, create a virtual environment in Python, and install the packages in this requirements file.

## Bibliometric data on subnational internal and international flows Scholarly Migration Database 2.0

See the website of the Scholarly Migration Database for the subnational data as `rates` (migration measures) and `bilateral flows` [https://www.scholarlymigration.org/](https://www.scholarlymigration.org/). Here on Zenodo, you can download the data [https://doi.org/10.5281/zenodo.20487917](https://doi.org/10.5281/zenodo.20487917).
