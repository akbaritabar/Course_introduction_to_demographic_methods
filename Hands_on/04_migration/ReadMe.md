## Simulated and synthetic data

When you use data that is not publicly shareable due to data license issues, personal data and re-identification risks for individuals, and so on, you can use synthetic and simulated data that would allow you to share your scripts and this `fake` data and if one could access the data on their own, they can use your scripts to replicate the analysis.

In the [second Rostock Open Science workshop at MPIDR in 2025](https://www.demogr.mpg.de/en/news_events_6123/calendar_1921/second_rostock_open_science_workshop_13282), Lisa DeBruine (University of Glasgow) gave a workshop on "Replicability of results in the context of private non-sharable data" and available R packages. Her slides and materials are available (as of today, 2026-01-14) at this link [https://debruine.github.io/talks/rostock-datasim/](https://debruine.github.io/talks/rostock-datasim/) (let me know if you had issues downloading them).

Check her slides for the required R packages and use `install.packages('package-name')` to install them.

## Bibliometric data on subnational internal and international flows in "Subnational_flow_data_Scopus_2020"

The folder `Subnational_flow_data_Scopus_2020` includes data from this paper [https://doi.org/10.1073/pnas.2424521122](https://doi.org/10.1073/pnas.2424521122). Please read the readme file in that folder [./Subnational_flow_data_Scopus_2020/ReadMe.md](./Subnational_flow_data_Scopus_2020/ReadMe.md) carefully on instructions how to use the data. Please do not share this data as I am only including them to allow you to use it in your final essay, if interested.
