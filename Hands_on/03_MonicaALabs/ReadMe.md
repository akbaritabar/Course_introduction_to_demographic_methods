## Data to download

Below, I have included the lab session script names and the **data files that you should download on your own** as I could not upload them due to file size limits. Please note the version of data and year, in case they are updated.

**Please note**: Include the downloaded files in the `data` folder as scripts read from there.

### "1_intro_edits.qmd"

To run "1_intro_edits.qmd", you need to visit the **World Population Prospects 2024** at this link [https://population.un.org/wpp/downloads?folder=Standard%20Projections&group=Population](https://population.un.org/wpp/downloads?folder=Standard%20Projections&group=Population) and download the following files:

1. Single Ages, Population by Single Age - Male (XLSX): https://population.un.org/wpp/assets/Excel%20Files/1_Indicator%20(Standard)/EXCEL_FILES/2_Population/WPP2024_POP_F01_2_POPULATION_SINGLE_AGE_MALE.xlsx
2. Single Ages, Population by Single Age - Female (XLSX): https://population.un.org/wpp/assets/Excel%20Files/1_Indicator%20(Standard)/EXCEL_FILES/2_Population/WPP2024_POP_F01_3_POPULATION_SINGLE_AGE_FEMALE.xlsx
3. The following two files for mortality are already included in the `data` folder:
    1. "WPP2024_MORT_F01_2_DEATHS_SINGLE_AGE_MALE.xlsx"
    2. "WPP2024_MORT_F01_3_DEATHS_SINGLE_AGE_FEMALE.xlsx"
